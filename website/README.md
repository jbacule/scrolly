# Scrolly Website

This folder contains the Astro-based landing page for the Scrolly browser extension.

## Setup

```bash
cd website
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Update Content

- Main page: `website/src/pages/index.astro`
- Global styles: `website/src/styles/global.css`
- Images: `website/public/images/`
